Browser-only utility for Bluesky exports

Bluesky CAR Post Extractor

`.car` をブラウザで直接展開し、`createdAt` と `text` だけを含む JSON / CSV を生成します。 ファイルはアップロードされず、このページの中だけで処理されます。

Step 0

Bluesky アプリで `.car` を取得

2026-03-24 時点で確認できた Bluesky の現行 UI 文言ベースでは、`Settings` → `Account` → `Export my data` → `Download CAR file` の順です。

  1. Bluesky アプリまたは Bluesky Web を開いて `Settings` を開く。
  2. `Account` を開く。
  3. `Export my data` を選ぶ。
  4. `Download CAR file` を押して `repo.car` を保存する。
  5. 保存した `.car` をこのページに戻ってアップロードする。

Bluesky の説明では、この CAR には公開データの記録が含まれますが、画像などの media embeds や private data は含まれません。

Official blog: repo export API: com.atproto.sync.getRepo App source: AccountSettings App source: ExportCarDialog
